## Github Release 에서 업로드 하기
1. Github에 들어가서 Dspaut-download를 클릭함.
2. dspaut-download 이름 아래 main / branch / tag 중 tag 클릭함.
3. Tag 들어가면 1.0.0에 클릭함.
4. 업로드한 목록을 확인할 수 있고 여기서 새롭게 올리기 위해서는 오른쪽 연필 모양을 클릭함.
5. 클릭 후 추가로 작성하는 것은 없고 그냥 프로그램만 하단 업로드하는 곳에 이동 후 Update release를 클릭함.
6. 업로드 완료 후 설정 방법에 downloadUrl 주소는 Assets아래 업로드한 프로그램에 오른쪽 클릭 후 "링크 주소 복사"를 클릭함.
7. 클릭하면 주소가 나오고 downloadUrl 주소에 입력함.
